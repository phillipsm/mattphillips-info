an investigation of digital stereoscopic photography

About

We see in stereo. We have two eyes and each one captures its own image as we navigate the world. Our brains wire the two images together providing visuals rich with depth and texture.

What if our cameras could see the way we see? They can - we simply need to add lenses.

This is an investigation in how to build an open, affordable, digital stereoscopic camera.

Technology

I have created several stereoscopic camera prototypes. The latest uses four cameras, wired to an SD card, controlled by an Arduino microcontroller.

When the shutter release button is pressed, the camera writes four simutanously captured images to the SD card. From the SD card, I link the images toghether in Photoshop and save them as a six frame animated GIF.

Source

Source code is available through a dual MIT/GPL license on GitHub.
